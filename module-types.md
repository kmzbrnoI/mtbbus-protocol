MTBbus module types
===================

MTBbus module type is determined by its 1-byte *code*.

| Code   | Module type                   |
|--------|-------------------------------|
| `0x10` | MTB-UNI v2 with ATmega328p    |
| `0x15` | MTB-UNI v4 with ATmega128     |
| `0x20` | MTB-BOOST                     |
| `0x30` | MTB-RAILCOM                   |
| `0x40` | MTB-CROS                      |
